1. Install the following libaries via pip:
Levenshtein
openai
elevenlabs
(optional, used for the built-in modules)
geocoder

2. From \Nova, run StartInterface.py
3. Type 'keys' and enter your API keys. You need: OpenAI, Elevenlabs, OpenWeather(optional, is used for the weather module)
4. Type 'boot' to start Nova
